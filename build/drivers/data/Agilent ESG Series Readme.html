Agilent ESG Series Instrument Driver Readme

To rate this driver, go to the driver download page on the Instrument Driver Network.

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Agilent Technologies
Supported Language(s): LabVIEW
Supported Model(s): E4400B, E4420B, E4421B, E4422B, E4423B, E4424B, E4425B, E4426B, E4430B, E4431B, E4432B, E4433B, E4434B, E4435B, E4436B, E4437B, E4438C, E4428C
Model(s) Tested: E4438C
Interface(s): Serial, GPIB
Firmware Revision Tested: B.01.01

Certified: Yes
NI Supported: Yes
Source Code Available: Yes

Driver Revision: 3.0.4
Original Release Date: 01/2000
Current Revision Date: 08/20/2013

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

VISA 5.0 or later

For more details, refer to the LabVIEW Help under Help » VI, Function, & How-To Help.

3. Known Issues

To report issues or provide feedback about this instrument driver, please contact technical support. -Although support for the E4428C model has been added in this driver, it has not been tested. Refer to instrument documentation for more details for the E4428C model.
